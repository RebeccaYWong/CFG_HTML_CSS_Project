Online Resources

This page offers a compilation of useful information regarding free online resources to help you learn code.

  • Code Academy

    Code Academy allows you to code interactively for free. This online platform offers free coding classes in 12 different programming languages including Python, Java, PHP, JavaScript, Ruby, SQL, and Sass, as well as markup languages HTML and CSS.

    Expertise: No prior expertise needed

    Topics: HTML & CSS, JavaScript, PHP, Python, Ruby, Angularjs and more

  • Code Wars

    Code Wars offers an exciting learning experience through the use of coding games with a martial arts theme. There are also opportunities to interact with and collaborate with the CodeWars community.

    Expertise: No prior expertise needed

    Topics: JavaScript, Python, Ruby, Java, CoffeeScript, Haskell and Clojure

  • General Assembly

    General Assembly is a project-based online learning course that guides you though how to create your own website.

    Expertise: No prior expertise needed

    Topics: CSS and HTML, JavaScript

  • Free Code Camp

    Free Code Camp is a camp-themed online coding community provides an established programme, taking approximately 800 hours to complete.

    Expertise: No prior expertise needed

    Topics: CSS and HTML, Java, Git, Agile and more
